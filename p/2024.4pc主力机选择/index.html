# 零散

‍

生产力用户更偏重多核性能，可以参考综合性能天梯图
游戏用户更偏重单核性能，综合性能天梯图参考意义不大

‍

如果不考虑价格的问题，确实U7和U9在功耗给足的情况下，性能也着实要比8845/7840好一些。

可是，如果综合性能和价格的对比，我个人会更推荐AMD 8845或7840的机型，尽管性能比U7略弱一点点，但是通常搭载8845/7840的机型，同等配置下，要比U7版本便宜1000元左右，这也是AMD处理器的价格优势。 AMD yes!

因为我们在现实世界中，买到的笔记本电脑就是一台完整的、上手即用的普通消费级电子产品。目前同配置的机型下，比如小新Pro16，它的AMD锐龙机型就是要比酷睿Ultra便宜好几百，性能也是互有胜负的水平，因此目前7840以及后续的8845显然性价比都更高。

而且，8845/7840锐龙版现阶段还有个优点就是“可选择的产品非常多”。原因是7840上市到现在有半年时间了，8845也开始有不同厂家推出轻薄本可选，结合前文关于酷睿Ultra和8845/7840的性能对比，这些机型即使放到2024年甚至2025年都不过时。

‍

轻薄本Intel处理器优先推荐i5-13500H，4大核+8小核架构，采用12核16线程，最高睿频可达4.7GHz，大部分需要高性能处理器的需求都可以满足，没必要加钱去选择i7-13700H处理器。

‍

游戏本买来主要是创作设计、三维建模、深度学习等，选择Intel酷睿处理器，对Adobe全家桶软件优化比较好。处理器推荐选择i5-13500HX，游戏党高性价比处理器，采用6大核+8小核架构，三级缓存24MB，睿频可达4.6GHz。

AMD锐龙处理器价格一般比Intel酷睿版便宜，但是实际处理器性能差距不大，差价在300左右，选择Intel酷睿处理器，差价过大就选择AMD锐龙处理器，比如R7-7745HX、R9-7845HX，性价比更高。

如果游戏本买来只是玩大型3A游戏，而游戏性能表现瓶颈主要在于显卡，同样的显卡，AMD和Intel处理器的游戏本实际帧数表现几乎没有差别，按预算选择。(吕布骑狗)

# Intel处理器型号辨别

‍

Intel最新的处理器是13代酷睿，分为HX、H、P、U四个系列，代表着不同的性能，HX系列处理器支持超频，适合游戏发烧友，搭载在游戏本上。

H系列则是高性能高功耗，搭载在轻薄本或者游戏本上。P系列主要是追求性能和续航之间的平衡，搭载在轻薄本上，U系列处理器则是低性能，发热量低，用在二合一笔记本上。

Intel酷睿13代处理器定位有i3、i5、i7、i9，今年主流的处理器型号定位是i5和i9，i7夹在中间比较尴尬，处理器性能基本是溢出状态，大部分需求选择i5就够了，追求极致的处理器性能释放，就选择i9。

比如轻薄本上选择i5-1340P、i5-13500H处理器就够用了，游戏本上就选择i5-13500HX、i9-13900HX，对于i7-13700H、i7-1360P处理器，一般是价格偏高，相比顶配i9没有什么价格优势

​​

‍

# AMD处理器型号辨别

关键点就是看处理器型号的第三个数字，代表着处理器型号所使用的架构，AMD锐龙最新的处理器架构是Zen4，那么处理器型号第三个数字是4的，就是最新架构处理器，比如R9-7940H、R9-7845HX。

如果数字是3，那就是旧架构，Zen3+、Zen3，这都是去年发布过的处理器，今年是重新炒冷饭，CPU规格几乎没有变化，改了个名字，比如R7-7735H，其实就是去年的R7-6800H。

‍

​​

‍

看完架构，看AMD锐龙系列，分为7045、7040、7035、7030、7020五个系列，加粗的数字都是代表它们的架构，重点看7045、7040系列处理器就可以，其它都是炒冷饭。

7045系列处理器后缀都是HX，专门搭载在游戏本上，最高16核32线程，适合游戏性能发烧友，可以超频。推荐R7-7745HX、R9-7945HX处理器型号。

7040系列处理器后缀则是H，搭载在轻薄本或者游戏本上，这个系列处理器最适合搭载在轻薄本上，因为拥有笔记本电脑上最强的核显，满血Radeon 780M相比上代强10%左右，比如R9-7940H、R7-7840H处理器，而R5-7640H核显是残血，就不推荐选择了。

# 显卡

见哔站收藏夹

选择中端卡即可, 不需要玩太多游戏, 3A能摸到就好, 4050(3060)的标准及以下一点

‍

中端卡：6650XT/7600/4060/6750GRE

‍

6650XT – 大于3060, 在没TI和有TI之间, 目前游戏卡性价比最高的选择，再往下都比较拉跨，再往上相对性价比又不及6650XT，1080P 3A游戏都能高画质畅玩，如果不考虑高分辨率那这张卡就挺合适的。

6750GRE – 对标3060TI 4060的价格，4060Ti的性能，2000左右游戏性价比最高的卡，有10GB和12GB两个版本，买12GB的。

4060 – 大于3060, 在没TI和有TI之间N卡特有的优势，功耗低，游戏性能在2K基本都能中高画质玩，1080P基本通杀。

‍

显卡功耗不明确的

详情页没有明确标识显卡功耗或者语焉不详的，不用想就知道是残血版。如果是游戏玩家推荐满血版显卡，主要当做生产力的话残血版可以考虑。

‍

入门卡：3050和6500XT(<3050)

这两个也基本确保无矿，但是3050价格太高，性价比太低，6500XT虽然性价比也低，但是相对高一些，都不是特别推荐。

‍

‍

RTX4060相比RTX3060性能提升15%左右，能耗提升较大，功耗只需要达到110W左右，就可以跑满频率，对游戏本散热能力要求更低。

如果RTX4060和RTX4050显卡版本价格差距只有500左右，就可以获得21%左右的性能提升，直接选择RTX4060版本，性价比更高。

RTX4070是RTX40系中性价比最低的一款显卡型号，不推荐选择，相比RTX4060性能提升不大，差价却基本在3000左右，性价比很低。

‍

那么我会选用4050/60

‍

‍

2023年笔记本电脑显卡更新到了RTX40系，MX系列独显已经推出了历史舞台，只剩下RTX系列。

2023年RTX40系新显卡是RTX4050、RTX4060、RTX4070、RTX4080、RTX4090，性能由低到高排列。

同时部分轻薄本还会继续用去年发布的RTX3050、RTX2050显卡，比如2023年新增的6GB显存版本的RTX3050，还有小封装版64bit的RTX3050，以下图表标注为黄色名称。

其中64bit版本的RTX3050 4GB其实和RTX2050性能基本一样，所以搭载这款独显的价格比核显版贵1000左右比较合理，差价过多就不推荐入手。

RTX40系显卡能效提升，所以这一代的显卡用起来更省电，也不再需要满血功能就可以发挥全部性能，比如95W的RTX4050显卡性能和140W的差不多，110W的RTX4060和140W的差不多。

能效的提升，对于全能本来说意义很大，即使搭载残血RTX40系显卡，也可以拥有不错的显卡性能释放，这简直就是全能本的春天。

​​

综合来说，如果RTX4050和RTX4060游戏本价格差距只有500，直接上RTX4060，显卡性能直接获得22%左右的提升。如果差价在1000左右，那就看预算和需求了。

RTX4070是RTX40系显卡中性价比最低的一款，不推荐大家选择RTX4070游戏本，因为RTX4070性能相比RTX4060只提升14%左右，但是价格却相差两三千，智商检测卡。

‍

# 内存

笔记本电脑内存容量都是16G起步，16G内存容量对于大多数人来说已经够用了，如果有开虚拟机、运行大量代码、跑仿真等使用需求，选择32G或者支持内存拓展的笔记本电脑。注意大多数轻薄本内存是板载设计，无法升级。

笔记本电脑内存看内存类型和内存频率，最新内存类型是DDR5，相比DDR4内存性能更强，但是前期内存延迟比较高，后期优化好之后，DDR5内存对游戏性能也可以带来一点提升。

‍

‍

# 硬盘

需要注意的就是硬盘拓展性，看硬盘有几个M.2插槽，单M.2硬盘插槽，硬盘容量升级只能将原来的硬盘替换掉，升级成本高。双M.2硬盘插槽，就是还有一个硬盘位可以自己加装硬盘。
对于硬盘支持拓展的笔记本电脑，需要注意可拓展的尺寸，有2242和2280规格，2280的固态硬盘比较多，选择很多。2242的固态硬盘比较少，不好买。

‍

‍

# 接口

  • USB-A：主流是USB3.0和USB2.0。USB3.0传输速率更快，经常用于连接鼠标、U盘等。
  • RJ-45：以太网卡接口，就是用来插网线的。
  • HDMI：用来连接投影仪、显示器，办公族、老师等通常会用到这个。
  • Type-C：正反都可以插，可以用来传输数据、快速充电等，未来接口发展方向。
  • 耳麦接口：用来插耳机、麦克风。

‍

有无雷电接口

雷电是intel的专利，所以只有搭载酷睿处理器的机型有，AMD的机器没有，且同系列的机型采用酷睿处理器一般要更贵些。

最新的雷电四接口传输速率最高可达40Gbps(5GB/s)，可以外接4K或8K的显示器。 接口这种东西很神奇，如果你不是常用的，大概率到笔记本退休你都用不了几次；

如果是要经常外接显示器玩游戏、视频剪辑之类的，最好选择有雷电接口的酷睿处理器；如果雷电接口不是刚需，考虑性价比推荐锐龙处理器。

确定该如何选择
