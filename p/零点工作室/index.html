零点工作室

2022-12-19
阅读时长: 2 分钟

# 起因

和上次的动漫社事情一样, 是学校要求必须要报社团才去的, 之前没太多想法

当时宿舍全员说都报一个社, 能进就都进, 那时我们还是信心十足的呢

选择报名之后才知道原来需要面试才能进, 那就面吧.

# 面试大杀特杀

当时是 10 月份的一个星期四, 面试人很多, 都在主楼 4 楼那一排站着 分笔试和面试. 轮到我进去后, 先领取笔试卷子, 框框写, 记得就是一些基础的 Cpp 题目和高考数学题 (我不知道为什么要做数学题, 感觉没什么意思); 之后就是面试, 会询问一些软件专业相关的基础问题 (cpp) 还会问一些 python 相关的东西, 因为我主动说我会 Python, 但是他问我一些特殊的 Python 语法糖, 我当然那个时候是没记住的, 我觉得不需要…直接补全就好了

于是我只能回答 “哦, 这个只是一些基础逻辑的掌握, 我还做了…”, 现在发现, 哦这就是我的第一次面试, 第一次感受到了"博弈".

我记得那次我拿出了所有本领, 框框说, 有点对不住和我一起来的舍友, 形成对比了哈哈 (sorry man, if u are watching)

然后就是等通知

# 不合适就别强求

一周后一个宿舍就我进去了, “被大伙孤立”, 其实我心里暗爽.

进去了之后呢, 也就做一些简单的集体训练, 学习 Cpp 特性, 学习 HTML + CSS, 学习 DreamWeaver 这样的设计, 但是实际上里面就是搬运一些菜鸟和一些博客, 就是照本宣科, 没什么技术实力. 只能说是给了自己一个"必须要学习的理由"

我很反感这样的活动, 于是在 12 月退出了. 我完全不需要别人督促我学习, 我希望学习是一个主动的过程, 这样的学习风格我一直以来都很不爽.

从内到外都充满了一种 “衡水味”, 我在高中就相当反感的东西, 那种指着你脑袋要你学 + 填鸭式教育的组合, 让我完全没有兴趣. 浪费了 2 周时间去完成他们无理由的作业, 实在是无趣至极.

所以其实我想, 当初就不要通过为好, 浪费时间. 当然现在看来, 至少获得了一些聊以慰藉的"面试经验"与"社团经验"罢了.

我不知道其后的社团活动内容, 但是从一开始我就知道 “这不适合于我”, 那么我就会退出, undoubtedly. 之后的青城培训也是一样, 既然不合适就不要强求自己, 哥们应润尽润.

就这样.
